================================================================================
MMAP WORKFLOW
================================================================================

1. File on Disk
   в””в”ҖвҶ’ open() + flock() (POSIX) or CreateFile() (Windows)
   
2. Memory Map
   в””в”ҖвҶ’ mmap() (POSIX) or MapViewOfFile() (Windows)
       вҖў Read-only access (PROT_READ / FILE_MAP_READ)
       вҖў Private mapping (MAP_PRIVATE on POSIX)
       вҖў Sequential hint (POSIX_MADV_SEQUENTIAL on POSIX)
   
3. Pointer to Weight Data
   в””в”ҖвҶ’ data = (tn_i8*)mf.data + 64
       вҖў mf.data = pointer to start of file
       вҖў +64 = skip past header/config/padding
   
4. weights_map() Execution
   в””в”ҖвҶ’ Direct pointer arithmetic (NO fseek)
       вҖў ptr starts at data
       вҖў Each MAP_PTR advances ptr by read size
       вҖў Each ALIGN64 adds padding
       вҖў All bounds checked

5. Result
   в””в”ҖвҶ’ TransformerWeights struct populated with pointers
       вҖў All pointers point into mmap'd region
       вҖў Direct memory access, no copying
       вҖў On process exit, munmap() releases mapping

READING SEQUENCE FLOWCHART
================================================================================

  START (weights_map called)
      в”Ӯ
      в”ңв”ҖвҶ’ ptr = data
      в”ңв”ҖвҶ’ Extract config: dim, hidden_dim, kv_dim, n_layers
      в”Ӯ
      в”ңв”ҖвҶ’ [STEP 1] READ TOKEN EMBEDDING TABLE
      в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(token_embedding_table, vocab_size * dim * 4)
      в”Ӯ   в”ңв”ҖвҶ’ ALIGN64()
      в”Ӯ   в””в”ҖвҶ’ token_embedding_scale = 1.0f
      в”Ӯ
      в”ңв”ҖвҶ’ FOR LAYER l = 0 to n_layers-1:
      в”Ӯ   в”Ӯ
      в”Ӯ   в”ңв”ҖвҶ’ [STEP 2] INPUT ATTENTION NORM
      в”Ӯ   в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(rms_att_weight[l], dim * 4)
      в”Ӯ   в”Ӯ   в””в”ҖвҶ’ ALIGN64()
      в”Ӯ   в”Ӯ
      в”Ӯ   в”ңв”ҖвҶ’ [STEP 3] POST-ATTENTION NORM
      в”Ӯ   в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(rms_ffn_weight[l], dim * 4)
      в”Ӯ   в”Ӯ   в””в”ҖвҶ’ ALIGN64()
      в”Ӯ   в”Ӯ
      в”Ӯ   в”ңв”ҖвҶ’ [STEP 4] QUERY (wq + sq)
      в”Ӯ   в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(wq[l], packed_bytes(dim*dim))
      в”Ӯ   в”Ӯ   в””в”ҖвҶ’ COPY_VAL_ALIGN(&sq[l], 4)
      в”Ӯ   в”Ӯ
      в”Ӯ   в”ңв”ҖвҶ’ [STEP 5] KEY (wk + sk)
      в”Ӯ   в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(wk[l], packed_bytes(dim*kv_dim))
      в”Ӯ   в”Ӯ   в””в”ҖвҶ’ COPY_VAL_ALIGN(&sk[l], 4)
      в”Ӯ   в”Ӯ
      в”Ӯ   в”ңв”ҖвҶ’ [STEP 6] VALUE (wv + sv)
      в”Ӯ   в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(wv[l], packed_bytes(dim*kv_dim))
      в”Ӯ   в”Ӯ   в””в”ҖвҶ’ COPY_VAL_ALIGN(&sv[l], 4)
      в”Ӯ   в”Ӯ
      в”Ӯ   в”ңв”ҖвҶ’ [STEP 7] ATTENTION SUB-NORM (BitNet-specific)
      в”Ӯ   в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(rms_attn_sub_norm[l], dim * 4)
      в”Ӯ   в”Ӯ   в””в”ҖвҶ’ ALIGN64()
      в”Ӯ   в”Ӯ
      в”Ӯ   в”ңв”ҖвҶ’ [STEP 8] OUTPUT PROJECTION (wo + so)
      в”Ӯ   в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(wo[l], packed_bytes(dim*dim))
      в”Ӯ   в”Ӯ   в””в”ҖвҶ’ COPY_VAL_ALIGN(&so[l], 4)
      в”Ӯ   в”Ӯ
      в”Ӯ   в”ңв”ҖвҶ’ [STEP 9] FFN GATE (w1 + s1)
      в”Ӯ   в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(w1[l], packed_bytes(dim*hidden_dim))
      в”Ӯ   в”Ӯ   в””в”ҖвҶ’ COPY_VAL_ALIGN(&s1[l], 4)
      в”Ӯ   в”Ӯ
      в”Ӯ   в”ңв”ҖвҶ’ [STEP 10] FFN UP (w3 + s3)
      в”Ӯ   в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(w3[l], packed_bytes(dim*hidden_dim))
      в”Ӯ   в”Ӯ   в””в”ҖвҶ’ COPY_VAL_ALIGN(&s3[l], 4)
      в”Ӯ   в”Ӯ
      в”Ӯ   в”ңв”ҖвҶ’ [STEP 11] FFN SUB-NORM (BitNet-specific)
      в”Ӯ   в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(rms_ffn_sub_norm[l], hidden_dim * 4)
      в”Ӯ   в”Ӯ   в””в”ҖвҶ’ ALIGN64()
      в”Ӯ   в”Ӯ
      в”Ӯ   в””в”ҖвҶ’ [STEP 12] FFN DOWN (w2 + s2)
      в”Ӯ       в”ңв”ҖвҶ’ MAP_PTR(w2[l], packed_bytes(dim*hidden_dim))
      в”Ӯ       в””в”ҖвҶ’ COPY_VAL_ALIGN(&s2[l], 4)
      в”Ӯ
      в”ңв”ҖвҶ’ [STEP 13] FINAL LAYER NORM
      в”Ӯ   в”ңв”ҖвҶ’ MAP_PTR(rms_final_weight, dim * 4)
      в”Ӯ   в””в”ҖвҶ’ ALIGN64()
      в”Ӯ
      в”ңв”ҖвҶ’ [STEP 14] OUTPUT CLASSIFIER (Tied Embeddings)
      в”Ӯ   в”ңв”ҖвҶ’ wcls = (tn_i8*)token_embedding_table
      в”Ӯ   в””в”ҖвҶ’ wcls_scale = 1.0f
      в”Ӯ
      в””в”ҖвҶ’ RETURN TN_OK

  ERROR PATHS:
      Any bounds check fails вҶ’ RETURN TN_ERR_INVALID_WEIGHTS
      
================================================================================
BOUNDS CHECKING TREE
================================================================================

EVERY READ IS GUARDED:

    if ((size_t)(ptr - data + bytes_to_read) > data_size)
        return TN_ERR_INVALID_WEIGHTS;

Catches:
    в”ңв”ҖвҶ’ File truncated before completing read
    в”ңв”ҖвҶ’ Integer overflow in size calculation
    в”ңв”ҖвҶ’ Scale factor extends beyond buffer
    в””в”ҖвҶ’ Alignment padding extends beyond file

Example failure:
    rel_offset = 4227090
    bytes_to_read = 4 (scale factor)
    (4227090 + 4) > data_size  вҶ’  FAIL вң—
    
    Converter must account for every byte!

TERNARY PACKING DETAIL
================================================================================

INPUT: Array of ternary values {-1, 0, 1}
       Example for Query matrix: dim Г— dim = 16,777,216 values

PACKING:
    4 weights per byte, 2 bits each
    
    в”Ңв”Җ Byte (8 bits) в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
    в”Ӯ [7:6] [5:4] [3:2] [1:0]         в”Ӯ
    в”Ӯ  w[3]  w[2]  w[1]  w[0]         в”Ӯ
    в”Ӯ  2bit  2bit  2bit  2bit         в”Ӯ
    в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

    Encoding: value + 1 вҶ’ 2-bit result
    -1 вҶ’ 0b00
     0 вҶ’ 0b01
     1 вҶ’ 0b10
     (unused: 0b11)

    packed_bytes = ceil(16,777,216 / 4) = 4,194,304 bytes
    
    On disk after packing:
    в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ

    в”Ӯ 4 bytes (scale factor sq[l]) в”Ӯ
    в”ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Ө

UNPACKING (reference):
    tn_i8 unpack_ternary(const tn_u8 *packed, int index) {
        tn_u8 byte = packed[index >> 2];
        int shift = (index & 3) << 1;
        return (tn_i8)((byte >> shift) & 0x03) - 1;
    }
