================================================================================
2. PACKED TERNARY FORMAT
================================================================================

4 weights per byte, 2 bits each:
   -1 → 0b00
    0 → 0b01
    1 → 0b10

packed_bytes(count) = ceil(count / 4) = ((count + 3) >> 2)

Example:
   For Query matrix: dim × dim = 4096 × 4096 = 16,777,216 weights
   Packed bytes: ceil(16777216/4) = 4,194,304 bytes

================================================================================
3. OFFSET CALCULATION (64-BYTE ALIGNMENT)
================================================================================

NOT using fseek() - uses MMAP pointer arithmetic instead.

Alignment base = 64 (file offset where weights start)

For each read:
   relative_offset = ptr - data
   absolute_offset = relative_offset + 64
   padding_needed = (64 - (absolute_offset % 64)) % 64
   ptr += padding_needed

Example sequence:
   rel_offset=0:          abs=64   % 64 = 0   → padding=0
   rel_offset=16384:      abs=16448 % 64 = 0   → padding=0
   rel_offset=4227076:    abs=4227140 % 64 = 4 → padding=60

================================================================================
4. BOUNDS CHECKING (EVERY READ)
================================================================================

Before every read, validation check:
   if ((ptr - data + bytes_to_read) > data_size)
       return TN_ERR_INVALID_WEIGHTS

Catches:
   - File truncation
   - Overflow in size calculations
   - Invalid alignment padding
   - Any read beyond buffer

5. MEMORY MAPPING (NOT FILE I/O)
================================================================================

POSIX (Linux):
   int fd = open(path, O_RDONLY)
   flock(fd, LOCK_SH | LOCK_NB)  ← Prevent concurrent modification
   void *data = mmap(fd, ..., PROT_READ, MAP_PRIVATE, ...)
   posix_madvise(..., POSIX_MADV_SEQUENTIAL)

Windows:
   HANDLE hFile = CreateFileA(..., GENERIC_READ, ...)
   HANDLE hMapping = CreateFileMappingA(hFile, ..., PAGE_READONLY, ...)
   void *data = MapViewOfFile(hMapping, FILE_MAP_READ, ...)

Result: Direct memory access via pointers, NO seeking
